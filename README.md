# App Mobile CRUD

Aplicativo mobile desenvolvido em Java que realiza operações de 
Create, Read, Update e Delete (CRUD) para gerenciamento de dados.

## Funcionalidades

- Criar novos chamados
- Listar chamados criados
- Editar chamados existentes
- Excluir chamados
- Validação básica de campos
- Edição de perfil de usuário
- validação de senha

## Tecnologias utilizadas

- Java
- Android Studio
- XML

## Como executar

1. Clone o repositório
2. Abra no Android Studio
3. Aguarde a sincronização
4. Execute no emulador ou celular
